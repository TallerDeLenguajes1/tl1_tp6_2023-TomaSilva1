#### RESPUESTAS PUNTO 4

### ¿String es una tipo por valor o un tipo por referencia?

- String es un tipo por referencia.

### ¿Qué secuencias de escape tiene el tipo string?

- Las secuencias de escape son: 

    \'     Comilla simple.
    \"     Comilla doble.
    \\     Barra diagonal inversa.
    \0     NULL.
    \a     Alerta.
    \b     Retroceso.
    \f     Avance de página.
    \n     Nueva Línea.
    \r     Retorno de carro.
    \t     Tabulación horizontal.
    \v     Tabulación vertical.
    \u     Secuencia de escape Unicode (UTF-16).
    \U     Secuencia de escape UNICODE (UTF-32).
    \x     Secuencia de escape Unicode similar a "\U" exceoto con longitud variable.

### ¿Qué sucede cuando utiliza el carácter @ y $ antes de una cadena de texto?

- Cuando utilizamos @ antes de una cadena de texto, cuando se muestre el texto en pantalla, no interpretara las secuencias de escape.

- Se utliza $ antes de una cadena para la Interpolación de cadenas, para mejorar la legibilidad y el mantenimiento del codigo.
